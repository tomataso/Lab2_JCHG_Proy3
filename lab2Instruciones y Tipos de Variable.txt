–Crear un API de un CRUD de Productos con la siguiente información

•Id autogenerado
•Nombre (string)
•Descripción (string)
•Cantidad (int)
•Precio (int)


–Utilizar MySQL, JPA y Spring Boot
Operaciones a generar
1. Crear Producto
2. Consultar todo los productos
3. Consultar producto por Id
4. Modificar un producto
5. Eliminar un producto
6. Busqueda de producto por nombre

Entregar el proyecto de IntelliJ y la colección de Postman para probar el CRUD
